GITHUB PAGES: ПОШАГОВАЯ ИНСТРУКЦИЯ

ВЫБРАННЫЙ ВАРИАНТ: GitHub Pages

1. СОЗДАЙТЕ РЕПОЗИТОРИЙ
- Войдите на https://github.com
- Нажмите New repository
- Имя: math-lessons
- Видимость: Public
- Не добавляйте README, .gitignore или лицензию, потому что они уже есть в архиве
- Нажмите Create repository

2. ЗАГРУЗИТЕ ФАЙЛЫ БЕЗ GIT
- Распакуйте архив
- В пустом репозитории нажмите uploading an existing file
- Перетащите всё содержимое распакованной папки
- Важно: index.html должен быть в корне репозитория, а не во вложенной папке
- Нажмите Commit changes

3. ВКЛЮЧИТЕ GITHUB PAGES
- Откройте Settings
- В левом меню выберите Pages
- В Build and deployment выберите Source: GitHub Actions

4. ДОЖДИТЕСЬ ПУБЛИКАЦИИ
- Откройте вкладку Actions
- Выберите Deploy static site to GitHub Pages
- Дождитесь зелёной отметки

5. ОТКРОЙТЕ САЙТ
Адрес для репозитория math-lessons:
https://ВАШ-ЛОГИН.github.io/math-lessons/

6. УСТАНОВИТЕ НА ANDROID
- Откройте сайт в Chrome
- Меню → Установить приложение или Добавить на главный экран
- Запустите приложение с иконки
- После первой загрузки все уроки доступны офлайн

ОБНОВЛЕНИЕ
- Загрузите изменённые файлы в репозиторий
- Commit changes
- Workflow повторно опубликует сайт автоматически

ЧАСТЫЕ ОШИБКИ
- 404: index.html загружен во вложенную папку, а не в корень
- Workflow не запускается: файлы отправлены не в ветку main
- Pages не публикуется: в Settings → Pages не выбран GitHub Actions
- Старая версия на телефоне: полностью закройте приложение и откройте снова; при необходимости очистите данные сайта
