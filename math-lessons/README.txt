МОДУЛЬНОЕ УЧЕБНОЕ ПРИЛОЖЕНИЕ

Почему нужен локальный сервер:
браузеры ограничивают загрузку JSON-файлов при открытии file://.

Windows:
1. Распакуйте архив.
2. Дважды нажмите start_windows.bat.
3. Откроется http://localhost:8000.

macOS / Linux:
1. Распакуйте архив.
2. В терминале выполните: sh start_mac_linux.sh

Структура:
- index.html: каталог уроков
- lesson.html: универсальная страница урока
- lessons/*.json: компактное содержание уроков
- js/tasks.js: единая фабрика задач
- js/demos.js: визуальные демонстрации
- js/app.js: навигация и рендеринг урока

ANDROID / PWA:
См. ANDROID_ИНСТРУКЦИЯ.txt. Добавлены manifest.webmanifest, sw.js и иконки. После установки по HTTPS приложение работает офлайн.
