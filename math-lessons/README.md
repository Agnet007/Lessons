# Математика от наблюдения к формуле

Интерактивное мобильное приложение с 15 уроками по функциям, геометрии и математике в физике.

## Быстрая публикация на GitHub Pages

1. Создайте на GitHub новый публичный репозиторий, например `math-lessons`.
2. Распакуйте этот архив.
3. Загрузите **всё содержимое папки** в корень репозитория. Файл `index.html` должен находиться в корне.
4. Откройте в репозитории `Settings` → `Pages`.
5. В разделе `Build and deployment` выберите `Source: GitHub Actions`.
6. Откройте вкладку `Actions` и дождитесь завершения workflow `Deploy static site to GitHub Pages`.
7. Сайт будет доступен по адресу:
   `https://ВАШ-ЛОГИН.github.io/ИМЯ-РЕПОЗИТОРИЯ/`

## Установка на Android

1. Откройте опубликованную HTTPS-ссылку в Chrome на Android.
2. Нажмите меню Chrome.
3. Выберите `Установить приложение` или `Добавить на главный экран`.
4. После первой полной загрузки приложение работает офлайн.

## Обновление

Измените нужные файлы и отправьте изменения в ветку `main`. GitHub Actions автоматически опубликует новую версию.

## Архитектура

- `index.html` — каталог уроков
- `lesson.html` — универсальная оболочка урока
- `lessons/*.json` — компактное содержание уроков
- `js/app.js` — навигация и рендеринг
- `js/tasks.js` — единая фабрика задач
- `js/demos.js` — визуальные демонстрации
- `sw.js` — офлайн-кэш PWA
- `manifest.webmanifest` — установка на Android
- `.github/workflows/deploy-pages.yml` — автоматическая публикация
