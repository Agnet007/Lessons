ОБНОВЛЕНИЕ СУЩЕСТВУЮЩЕГО РЕПОЗИТОРИЯ

1. Распакуйте архив.
2. В репозитории Lessons откройте папку math-lessons.
3. Удалите старые папки css, js, lessons, icons и старые файлы index.html, lesson.html, manifest.webmanifest, sw.js.
4. Загрузите вместо них содержимое папки math-lessons-v6.
5. Нажмите Commit changes.
6. Workflow Deploy static site to GitHub Pages запустится автоматически, потому что он уже находится в корне репозитория .github/workflows.
7. После зелёной галочки откройте https://agnet007.github.io/Lessons/
8. На телефоне полностью закройте установленное приложение и откройте снова. Если старая версия сохранилась, очистите данные сайта для agnet007.github.io и переустановите приложение.
